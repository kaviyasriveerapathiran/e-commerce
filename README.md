# ShopEase 🛒

A simple, beginner-friendly E-Commerce portal built with **HTML, CSS, and JavaScript** — created as a learning/practice project.

## 🌐 Pages

| Page | File | Description |
|------|------|--------------|
| Home | `assign3.html` | Landing page with welcome banner |
| About | `about.html` | Information about ShopEase |
| Products | `products.html` | Orders, Wishlist, Reviews, and Customer Care sections |
| Register | `register.html` | User registration form with validation |

## ✨ Features

- Responsive navigation bar across pages
- Order history display
- Wishlist section
- Customer reviews & ratings
- Customer care contact info
- "Report an Issue" button
- Form validation for registration (and login, via `script.js`)

## 🛠️ Tech Stack

- **HTML5** – Page structure
- **CSS3** – Styling (`styles1.css`)
- **JavaScript** – Form validation logic (`script.js`)
